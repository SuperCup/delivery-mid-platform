# 结算管理追踪系统

## 项目说明
本系统用于管理和追踪结算项目的进度，提供完整的结算流程管理功能。

## 项目结构
```
├── README.md              # 项目说明文档
├── docs/                  # 文档目录
│   └── layout-spec.md     # 布局规范文档
├── *.html                 # 页面文件
└── assets/               # 资源文件
```

## 布局规范
系统采用统一的布局规范，详细规范请参考 [布局规范文档](docs/layout-spec.md)。

## 开发指南
1. 所有新页面开发请严格遵循布局规范文档
2. 使用Tailwind CSS进行样式开发
3. 保持代码结构清晰，添加必要的注释

## 技术栈
- HTML5
- Tailwind CSS
- Font Awesome 图标库
